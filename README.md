## 🐝 Проект 1: «BeeGuardian» / «ПчелоСтраж»

**Краткая суть:**  
Сервис для защиты пчел от пестицидов при обработке полей. Telegram-бот оповещает пасечников о химических обработках, помогает фермерам соблюдать закон и дает «зеленый сертификат» для эко-продукции.

**Ключевая проблема:**  
Ежегодно в России гибнет **20–50 тысяч пчелосемей** из-за того, что фермеры не предупреждают о химобработках вовремя.

👉 [Читать полный бизнес-план →](ideas/BeeGuardian.md)

## 📚 Проект 2: Сервис дистрибьюции научных статей по предпочтениям

**Краткая суть:**  
Умная лента научных публикаций, которая изучает твои интересы и показывает только релевантные статьи из arXiv, журналов и Telegram-каналов. Никакого спама — только то, что действительно нужно.

**Ключевая проблема:**  
Исследователи тонут в потоке статей, тратят часы на фильтрацию и рискуют пропустить важные открытия в своей области.

👉 [Читать полный анализ проекта →](ideas/ScientificArticleDistr.md)

## Проект 3: Аэрофотосъёмка дачных участков

**Краткая суть:**  
Планировка участка с высоты птичьего полета. Заказываешь съёмку с дрона — получаешь 3D-модель и рекомендации, где поставить дом, чтобы не затенять грядки, и где проложить дорожки.

**Ключевая проблема:**  
С земли невозможно оценить реальные пропорции участка → ошибки в расположении построек, тень от дома убивает урожай, дорожки «съедают» место.
